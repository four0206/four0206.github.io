  1. Ques: What are the three componets of Web?

  2. Ques: Define version control and give one example.

    Ans: Version control is a system that records changes to a file or a set of files over time so that it is possible to easily recall specific versions later. One example of a version control is GitHub.

  3. Ques: What tag is used to create the title of a web page? Give an example.

  5. Ques: Identify the odd one out from: <span>, <p>, <a>, and <i> and state why.

    Ans: <p> is the odd one out because the rest are all inline elements while <p> is a block level element.

  6. Ques: What are the tags required to create a navigation list in HTML?

    Ans: <nav> then <ol> OR <ul> then <li>

  7. Ques: What element is used to create images? Name its attributes.

    Ans: <img>, its attributes are: src= (for the image source), alt= (for the alternate text), width= and height= (for rendering width and height without CSS)

  8. Ques: Identify the odd one out from these tags: <table>, <tr>, <b>, <td> and state the reason.

    Ans: The odd one out is <b>, because the others are all elements used for creating tables, while <b> simply bolds your text.

  9. Ques: Give an example of a color picker input.

    Ans: <input type="color" name="color" id="color" value="#FFC0CB">

  10. Ques: What is the use of name and value attributes? Give two examples.

    Ans: The name and value attributes work together. The name attribute can be used specify a name for an element, or it can also be used as a reference when data is submitted (e.g., from a form). The value attribute specifies the initial value of the element, so that even if two elements have the same name, they might not have the same value. In forms, the value attribute is what gets sent to the server--for example, if an elements name is "FName" for first name, the value sent to the server would be the first name a user specifies. Another example, also found in forms, would be a drop-down menu, where the entire menu is under the same name attribute (e.g., "homeprov" for "home province") while each option has its own value (e.g., "on" for "Ontario").

  11. Ques: Give an example of inline style.

    Ans: <style="color: pink">

  12. Ques: How would you link an external style sheet? Give an example.

    Ans: <link rel="stylesheet" type="text/css" href="css/style.css">

  13. Ques: Create a CSS block to center an element both horizontally and vertically that has position: absolute and has width: 200px and height: 200px.

    Ans:
    <style type="text/css">
     .container {
      position: absolute;
      left: 50%;
      top: 50%;
      width: 200px;
      height: 200px;
      margin:-100px 0 0 -100px;
     }
    </style>

  14. Ques: Create a CSS keyframes block called "spin" that rotates an element.

    Ans:
    <style type="text/css">
     @keyframes spin {
       from {
      transform: rotate(0deg);
       }
       to {
      transform: rotate(360deg);
       }
     }
    </style>

  15. Ques: Create a CSS animation block to rotate an element that has the following properties:

    • class: "rotate-me"
    • It rotates 5 times at intervals of 0.5s with a "linear" transition using the animation keyframes "spin"

    Ans:
    <style type="text/css">
     .rotate-me {
      animation:
       spin 0.5s linear 5;
      }
      @keyframes spin {
        from {
       transform: rotate(0deg);
        }
        to {
       transform: rotate(360deg);
        }
     }
    </style>

  16. Ques: How would you include/create an alert with string "I love JavaScript" using JavaScript?
