    Ans: Version control is a system that records changes to a file or a set of files over time so that it is possible to easily recall specific versions later. One example of a version control is GitHub.

    Ans: <img>, its attributes are: src= (for the image source), alt= (for the alternate text), width= and height= (for rendering width and height without CSS)
